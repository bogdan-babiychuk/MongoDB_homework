# MongoDB_homework

Итоговое задание по дисциплине «Программная инженерия 2»: проектирование
базы данных в MongoDB для учёта оценок студентов университета.

Подробный отчёт — в [REPORT.md](REPORT.md).

## Быстрый старт

```bash
# 1. Поднять MongoDB (если нет локально)
docker run -d --name mongo-hw -p 27017:27017 mongo:7

# 2. Установить зависимости
poetry install

# 3. Применить схемы, индексы, заполнить тестовыми данными и прогнать все 12 запросов
poetry run python -m src.main
```

## Структура

```
src/
├── db.py        — подключение к MongoDB
├── schema.py    — JSON Schema валидаторы коллекций
├── indexes.py   — вторичные индексы
├── seed.py      — генерация тестовых данных через Faker
├── queries.py   — 12 типовых запросов
└── main.py      — точка входа
```
